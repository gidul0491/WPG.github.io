시멘틱 태그

종류 설명
<header> 사이트 상단 혹은 좌측에 위치하며 검색 창이나 사이트 메뉴를 삽입
<nav> 문서 내의 다른 위치, 혹은 다른 웹 문서로 연결하는 링크를 포함
<main> 웹 문서가 보여주는 핵심 컨텐츠
<article> 문서 내의 독립된 컨텐츠 - article 하나만으로 완성된 컨텐츠를 이룬다는 의미
<section> 몇가지 내용을 묶어놓은 컨텐츠
<aside> 본문 내용 외의 좌,우 또는 하단에 만들어지는 사이드바
<footer> 웹 문서의 맨 아래쪽에 있는 영역으로 사이트 제작 정보, 저작권 정보, 연락처 등을 포함
<figure> 그림, 그래프, 사진같은 독립적인 내용
<figcaption> figure 태그의 제목
<summary> detail 태그의 보여지는 제목
<details>
숨길 수 있는 내용

details 태그 내의 summary 태그를 클릭하면 details 내의 나머지 요소들이 보이게 됨

텍스트

종류 설명
<h1~6> 제목(heading), 숫자가 클수록 크기가 작음
<p> 단락(paragraph), 앞 뒤로 빈 줄이 생겨 단락이 되는 텍스트
<blockquote> 인용문, 다른 텍스트보다 들여 써지며 화면 낭독기에서도 구분이 됨
<strong> 굵은 글씨로 강조, 화면 낭독기에서 구분이 됨
<bold> 굵은 글씨, 화면 낭독기에선 구분되지 않음
<em> 강조(emphasis), 글씨가 기울여지며 강조하고 싶은 부분에 사용, 화면 낭독기에서 구분이 됨
<abbr>
약자(abbreviation), 단어 위에 마우스 커서를 올리면 세부 정보(title="~" 속성)가 툴팁으로 표시됨
예시)
<abbr title="Hyper Text Markup Language">HTML</abbr> HTML
<cite>
인용, 기울여진 글씨체로 표시됨, 저작물의 출처를 표기할 때 사용하며 제목을 반드시 포함해야 함
예시)
<cite><a href="http://www.george-orwell.org/1984/0.html">Nineteen Eighty-Four</a></cite>
Nineteen Eighty-Four by George Orwell
<code>
컴퓨터 인식을 위한 소스코드
예시)
<code>hello world</code>
hello world
<small> 부가 정보처럼 작게 표시되는 텍스트
<sub> 아래 첨자(subscript)
<sup> 위 첨자(superscript)
<s> 취소선
<u> 밑줄 (underline)
<ins>
공동 작업 문서에서 새로운 내용을 삽입(insert), 속성으로 cite, datetime을 가질 수 있음
예시)
<ins cite="URL" datetime="YYYY-MM-DDThh:mm:ssTZD"></ins>
TZD(Time Zone Designator) like 2023-03-19T22:30:58Z(Z: 그리니치 표준시)
<del>
공동 작업 문서에서 기존 내용 을 삭제(delete), 속성으로 cite, datetime을 가질 수 있음
예시)
<del cite="URL" datetime="YYYY-MM-DDThh:mm:ssTZD"></del>
TZD(Time Zone Designator) like 2023-03-19T22:30:58+09:00(표준시 기준 +9시간, 한국 표준시)

목록

종류 설명
<ol> 순서가 있는 목록(ordered list), 속성으로 type="1 , a , A , i or I etc" 를 가질 수 있음
<ul> 순서가 없는 목록(unordered list)
<li> <ol>, <ul> 태그의 자식 요소로 사용되는 리스트태그
<dl> 설명목록(definition list)
<dt> 설명할 용어 이름(term)
<dd>
단어의 설명(description)
예시)
<dl>
  <dt>HTML</dt>
  <dd>Hyper Text Markup Language</dd>
</dl>
HTML

표

종류 설명
<table> 표의 시작과 끝을 알려주는 태그, border='1' 속성으로 테두리 지정 가능
<caption> 표의 제목
<tr> <table>태그 안에 사용되며 열(row)의 시작과 끝을정해주는 태그
<th> <tr>태그 안에 사용되며 데이터의 제목 등을 굵은 글씨로 나타냄
<td> <tr>태그 안에 사용되며 하나의 데이터 셀(data cell)을 정의
<thead> <table>태그 안에 사용되며 제목 등이 표시되는 표의 상단을 구분
<tbody> <table>태그 안에 사용되며 본문이 표시되는 부분
<tfoot> <table>태그 안에 사용되며 합계나 요약 등이 표시되는 하단을 구분
rowspan <td rowspan="합칠 셀의 개수"> 아래쪽 셀과 병합
colspan <td colspan="합칠 셀의 개수"> 오른쪽 셀과 병합
<colgroup> <caption> 태그 다음에 사용되며 테이블의 열을 묶어주는 태그
<col>
<colgroup>태그 내에 테이블의 열 개수만큼 사용되며 열 하나를 지정
예시)
<table>
  <caption>제목</caption>
  <colgroup>
    <col style="background-color: cyan;">
    <col style="background-color: lightcoral;">
  </colgroup>
...

멀티미디어

종류 설명
<img>
이미지를 삽입하는 태그
예시)
<img src="이미지 파일 경로" alt="대체용 텍스트" width="50%" height="100px">
<object>
오디오, 비디오, 자바 애플릿, PDF 등 멀티미디어 파일 삽입
예시)
<object width="50%" height="100px" data="파일경로"><object>
<embed>
오디오, 비디오, 이미지 등 멀티미디어 파일 삽입
예시)
<embed width="50%" height="100px" src="파일경로">
<audio>
오디오 파일 삽입
예시)
<audio src="파일경로"><audio>
<video>
비디오 파일 삽입
예시)
<video src="파일경로"><video>
controls 플레이어 화면에 컨트롤 바를 표시
autoplay 오디오나 비디오를 자동으로 실행
loop 오디오나 비디오를 반복 재생
muted 오디오나 비디오의 소리를 제거
preload
="auto, metadata or none" auto : 기본값, 페이지가 로드될 때 전체 비디오 파일도 같이 로드
metadata : 페이지가 로드될 때 비디오의 메타데이터만 로드
none : 페이지가 로드될 때 비디오가 로드되지 않음
width, height 비디오 플레이어의 너비와 높이 지정
poster="파일명" 비디오의 썸네일 이미지 설정
<a>
텍스트 또는 이미지를 링크로 만드는 태그 예시)
<a href="링크할 주소" target="_blank">텍스트 또는 이미지</a>

target="_blank" : 새 프레임에서 열기
_self : 기본값, 현재 프레임에서 열기
_parent : 현재 탭의 부모 프레임에서 열기
_top : 현재 윈도우 전체에서 열기
프레임 이름 : 명시된 프레임에서 열기

입력 양식

종류 설명
<form>
폼 요소를 감싸는 태그, 입력받은 자료를 서버에 어떻게 넘길 것인지 등을 설정 가능
method : 사용자가 입력한 내용을 서버로 어떻게 넘겨줄 것인지 지정 method="get" : 주소표시줄에 입력한 내용이 드러나고 256~4096 byte까지 서버로 넘길 수 있음
method="post" : 입력한 내용이 드러나지 않고 길이제한이 없음
name : 자바스크립트로 폼을 제어할 때 사용할 폼의 이름
action : <form> 태그 안의 내용을 처리해 줄 서버 프로그램을 지정
target : action 속성에서 지정한 스크립트 파일을 어느 위치에서 열것인지 지정
autocomplete : 자동완성기능 autocomplete="on" : 기본값, 자동완성기능 켜기
autocomplete="off" : 자동완성기능 끄기
<fieldset> <form> 태그 안에서 구역을 나누어 표시
<legend> <fieldset> 태그로 묶은 그룹에 제목 지정
<label>
<form> 의 요소에 레이블을 붙임 1. <label>레이블명<input ~></label>
2. <label for="id">레이블명</label>
   <input id="id" ~>
<input>
사용자가 데이터를 입력할 수 있도록 해주는 태그 autofocus : 페이지를 불러오자마자 해당 요소에 포커스를 맞춤
placeholder : 텍스트 등의 입력 필드에 힌트를 표시
readonly : 텍스트 등의 필드를 읽기 전용으로 만들어 수정할 수 없게 함
required : 필수 입력 필드로 지정
<textarea>
여러 줄을 입력하는 텍스트 영역 cols : 문자 개수를 단위로 하는 너비
rows : 즐 단위의 세로 길이
<select>
드랍다운 목록 size : 화면에 표시한 드랍다운 항목의 개수 지정
multiple : 둘 이상의 항목이 선택 가능한지 여부
<option>
<select> 태그의 자식 태그로 쓰이며 드랍다운 목록의 항목을 추가 <option value="값1">내용1</option>
<option value="값2">내용2</option>

value : 선택된 항목이 선택되었을 때 서버로 넘겨질 값
selected : 기본적으로 선택되어있을 항목
<datalist>
텍스트 필드에 데이터 목록을 만들어줌 예시)
<input type="text" list="데이터 목록 id">
<datalist id="데이터 목록 id">
  <option value="값1">내용1</option>
  <option value="값2">내용2</option>
</datalist>
<button>
버튼을 만들어주는 태그 <button type="속성"></button>
submit : 폼을 서버로 전송
reset : 폼에 입력한 내용 초기화
button : 버튼의 형태만 가지며 기능은 없음
onclick = "location.href = '#' " 링크 삽입 가능
<input> 태그의 type 속성
<input type="속성"> 설명
text
한 줄 짜리 텍스트 입력 창 size : text, password 필드의 길이(보이는 글자 수)
value : text 필드에 미리 보여주는 내용
maxlength : text, password 필드에 입력가능한 최대 문자 수
password 비밀번호 입력 창
search 검색 창
url URL 주소 입력 창
email email 입력 창
tel 전화번호 입력 창
checkbox
중복 체크가 가능한 체크박스 value : 선택한 checkbox나 radio 버튼을 서버에 넘겨줄때의 값
checked : checkbox, radio를 기본적으로 선택되어 있는 항목으로 설정
radio 1개만 선택가능한 라디오 버튼
number
숫자를 조절할 수 있는 스핀 박스 min : number, range 필드에서 설정할 수 있는 최소값, 기본값 0
max : number, range 필드에서 설정할 수 있는 최대값, 기본값 100
step : 숫자 간격, 기본값 1
value : 표시할 초기값
range 숫자를 조절할 수 있는 슬라이드 막대
date
사용자 지역을 기준으로 연, 월, 일 선택 min : 선택할 수 있는 날짜의 최소값
max : 선택할 수 있는 날짜의 최대값
step : 화살표를 눌렀을 때 이동하는 날짜의 크기
value : 표시할 초기값
month 사용자 지역을 기준으로 연, 월 선택
week 사용자 지역을 기준으로 연, 주 선택
time 사용자 지역을 기준으로 시, 분, 초, 분할 초를 선택
datetime 국제 표준시(UTC)로 설정된 연, 월, 일, 시, 분, 초, 분할 초 선택
datetime-local 사용자 지역을 기준으로 연, 월, 일, 시, 분, 초, 분할 초 선택
submit 전송 버튼 삽입
reset 리셋 버튼 삽입
image submit버튼 대신 사용할 이미지 삽입
button 버튼 삽입
file 파일을 첨부할 수 있는 버튼 삽입
hidden 사용자에게는 보이지 않지만 서버로 넘겨주는 값이 있는 필드 삽입
