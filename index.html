시멘틱 태그

종류 설명
<header> 사이트 상단 혹은 좌측에 위치하며 검색 창이나 사이트 메뉴를 삽입
<nav> 문서 내의 다른 위치, 혹은 다른 웹 문서로 연결하는 링크를 포함
<main> 웹 문서가 보여주는 핵심 컨텐츠
<article> 문서 내의 독립된 컨텐츠 - article 하나만으로 완성된 컨텐츠를 이룬다는 의미
<section> 몇가지 내용을 묶어놓은 컨텐츠
<aside> 본문 내용 외의 좌,우 또는 하단에 만들어지는 사이드바
<footer> 웹 문서의 맨 아래쪽에 있는 영역으로 사이트 제작 정보, 저작권 정보, 연락처 등을 포함
<figure> 그림, 그래프, 사진같은 독립적인 내용
<figcaption> figure 태그의 제목
<summary> detail 태그의 보여지는 제목
<details> 숨길 수 있는 내용, details 태그 내의 summary 태그를 클릭하면 details 내의 나머지 요소들이 보이게 됨

텍스트

종류 설명
<h1~6> 제목(heading), 숫자가 클수록 크기가 작음
<p> 단락(paragraph), 앞 뒤로 빈 줄이 생겨 단락이 되는 텍스트
<blockquote> 인용문, 다른 텍스트보다 들여 써지며 화면 낭독기에서도 구분이 됨
<strong> 굵은 글씨로 강조, 화면 낭독기에서 구분이 됨
<bold> 굵은 글씨, 화면 낭독기에선 구분되지 않음
<em> 강조(emphasis), 글씨가 기울여지며 강조하고 싶은 부분에 사용, 화면 낭독기에서 구분이 됨
<abbr> 약자(abbreviation), 단어 위에 마우스 커서를 올리면 세부 정보(title="~" 속성)가 툴팁으로 표시됨
예시)
<abbr title="Hyper Text Markup Language">HTML</abbr> HTML
<cite> 인용, 기울여진 글씨체로 표시됨, 저작물의 출처를 표기할 때 사용하며 제목을 반드시 포함해야 함
예시)
<cite><a href="http://www.george-orwell.org/1984/0.html">Nineteen Eighty-Four</a></cite> Nineteen Eighty-Four by George Orwell
<code> 컴퓨터 인식을 위한 소스코드
예시)
<code><p>hello world</p></code>

hello world
&lsmall> 부가 정보처럼 작게 표시되는 텍스트
<sub> 아래 첨자(subscript)
<sup> 위 첨자(superscript)
<s> 취소선
<u> 밑줄 (underline)
<ins> 공동 작업 문서에서 새로운 내용을 삽입(insert), 속성으로 cite, datetime을 가질 수 있음
예시)
<ins cite="URL" datetime="YYYY-MM-DDThh:mm:ssTZD"></ins>
TZD(Time Zone Designator) like 2023-03-19T22:30:58Z(Z: 그리니치 표준시)
<del> 공동 작업 문서에서 기존 내용 을 삭제(delete), 속성으로 cite, datetime을 가질 수 있음
예시)
<del cite="URL" datetime="YYYY-MM-DDThh:mm:ssTZD"></del>
TZD(Time Zone Designator) like 2023-03-19T22:30:58+09:00(표준시 기준 +9시간, 한국 표준시)
