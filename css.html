선택자

선택자
종류 설명
* { } 전체 선택자, 문서의 모든 요소에 스타일 적용
tagname { } 유형 선택자, 해당 태그에 스타일 적용
.classname { }
클래스 선택자, 해당 클래스에 스타일 적용 h1.text-black { } : .text-black 클래스를 가지는 h1태그를 선택
#id { } id 선택자, 해당 id에 스타일 적용
[ ] { }
속성 선택자, 주어진 속성을 가지는 모든 요소를 선택 h1[attribute] { } : attribute 속성을 가진 h1 요소를 선택
[attribute="value"] { }
속성의 값이 "value"인 요소를 선택 h1[title="abc"] : title="abc" 속성을 가진 h1 요소를 선택, title="abc def"는 선택되지 않음.
[attribute~="value"] { }
속성의 값이 "value"를 포함하는 요소를 선택 - 단어 기준 h1[title~="abc"] : title="abc def"는 선택되지만 title="abcdef"는 선택되지 않음.
[attribute|="value"] { }
속성의 값이 "value"이거나 "value-"로 시작하는 요소를 선택 h1[title|="abc"] : title="abc" || "abc-def는 선택되지만 title="abc def"는 선택되지 않음.
[attribute^="value"] { }
속성의 값이 "value"로 시작하는 요소를 선택 - 문자열 기준 h1[title^="abc"] : title="abc" || "abc-def || "abcdef" 모두 선택됨
[attribute$="value"] { }
속성의 값이 "value"로 끝나는 요소를 선택 - 문자열 기준 h1[title$="def"] : title="def" || "abc-def || "abcdef" 모두 선택됨
[attribute*="value"] { }
속성의 값이 "value"를 포함하는 모든 요소를 선택 - 문자열 기준 h1[title$="bc"] : title="abc" || "abc-def || "abcdef" 모두 선택됨
결합자
종류 설명
선택자1, 선택자2 { } 선택자 목록, 쉼표를 이용하여 여러 개의 선택자에 같은 스타일을 적용
선택자1 선택자2 { } 자손 결합자, 선택자 사이에 공백을 포함하여 선택자1의 자손인 선택자2에 스타일 적용
선택자1 > 선택자2 { } 자식 결합자, 선택자1의 자식 요소인 선택자2에 스타일 적용
선택자1 ~ 선택자2 { } 일반 형제 결합자, 선택자1을 뒤따르면서 같은 부모를 공유하는 형제요소인 선택자2에 스타일 적용
선택자1 + 선택자2 { } 인접 형제 결합자, 선택자1의 바로 뒤에 위치하는 형제요소인 선택자2에 스타일 적용
선택자1 || 선택자2 { } 열 결합자, 같은 열에 있는 요소를 선택
가상 선택자
종류 설명
::first-line { } 요소의 첫 번째 줄을 선택
::first-letter { } 요소의 첫 번째 문자를 선택
::before { } 요소의 앞을 선택
::after { } 요소의 뒤를 선택
::selection { } 마우스 드래그 등으로 선택한 텍스트를 선택
:link { } 방문하지 않은 링크에 스타일 적용
:visited { } 방문한 링크에 스타일 적용
:active { } 활성화된 웹 요소에 스타일 적용
:hover { } 마우스를 올린 상태의 요소를 선택
:target { } 앵커를 클릭해서 목적지로 이동했을 때, 목적지에 해당하는 요소에 스타일 적용
:enabled { } 해당 요소가 사용할 수 있는 상태일 때 스타일 적용
:disabled { } 해당 요소가 사용할 수 없는 상태일 때 스타일 적용
:checked { } 라디오 박스나 체크박스에서 선택된 항목에 스타일 적용
:not() { } 괄호 안에 있는 요소를 제외한 요소들에 스타일 적용
:empty { } 내용이 비어있는 요소를 선택, 공백은 내용이 있는 것으로 봄
:has( ) 괄호 안의 요소를 자식 요소로 가지는 부모 요소를 선택
A:only-child { } A 요소의 형제 요소가 없는 경우 해당 요소를 선택, A가 공백이면 종류에 관계없이 하나뿐인 요소를 선택
A:only-type-of { } 부모 안에 A타입 자식 요소가 하나 뿐일 때 A를 선택
A:first-child { } A와 형제인 요소 중 첫 번째 요소가 A일 경우에 선택, A가 공백이면 아무 첫 번째 요소 선택
A:last-child { } A와 형제인 요소 중에서 마지막 요소가 A일 경우 선택, A가 공백이면 아무 마지막 요소 선택
A:nth-child(n) { }
A의 형제 요소 중에서 특정 순서에 있는 요소가 A일 경우에 선택, A가 공백이면 아무 n번째 요소 선택 음이 아닌 정수를 뜻하는 n 사용 가능, 2n+1 혹은 even, odd 사용 가능
A:nth-last-child(n) { } A와 형제인 요소 중에서 뒤에서부터 n번째에 있는 형제 요소가 A일 경우선택, A가 공백이면 뒤에서 부터 아무 n번째 요소 선택
A:first-of-type { } A와 형제인 A 요소 중에서 첫 번째 요소를 선택
A:last-of-type { } A와 형제인 A 요소 중에서 마지막 요소를 선택
A:nth-of-type(n) { } A와 형제인 A 요소 중에서 n번째 요소 선택
A:nth-last-of-type(n) { } A와 형제인 A 요소 중에서 뒤에서 n번째 요소 선택

글꼴

종류 설명
font-family 글꼴 지정, 두 단어 이상으로 된 글꼴은 ""으로 표기, 첫 번째 글꼴이 지원되지 않는 경우를 대비하여 두개 이상의 글꼴 설정, 모든 자식 요소에 해당 글꼴 적용
font-size
글꼴 크기 지정 xx-small < x-small < small < medium < large < x-large < xx-large
em : 부모 요소에서 지정한 대문자 M 의 너비를 기준으로 한 상대적 크기 (1em = 16px = 12pt)
rem : 문서의 시작 부분에서 지정한 크기를 기준으로 한 상대적 크기
ex : 해당 글꼴의 소문자 x의 높이를 기준으로 한 상대적 크기
px : 모니터의 1픽셀을 기준으로 한 크기
pt : 포인트, 일반 문서에서 많이 사용
font-style
이탤릭체로 쓸지 말지를 결정 font-style : normal || italic || oblique
italic 체는 처음부터 디자인 된 글꼴인 반면 oblique는 현재 글꼴을 단순히 기울어지게 처리한 것
font-weight
글자 굵기를 지정, font-weight : normal || bold || bolder || lighter || 100 ~ 900 400 : normal
700 : bold
@font-face
폰트의 업로드와 임의의 폰트명으로 설정 후 지정 @font-face {
  font-family: '폰트 명'
  src: local('폰트파일이름') //사용자의 컴퓨터에 해당 폰트가 있는지 확인
  url('폰트파일 경로/.woff') format('woff') //업로드할 폰트 파일의 경로와 파일의 포맷
  url('폰트파일 경로/.ttf') format('truetype') //ttf는 truetype으로 입력
  url('폰트파일 경로.svg') format('svg')
}
.wfont {
  font-family: '폰트 명'; //웹 폰트 지정
}
@import url () fonts.google.com 에서 원하는 폰트를 찾고 @import url ( ) 부분을 복사한 뒤 style 태그에 붙여넣으면 폰트를 불러올 수 있음
color 글자색 지정, hsla(색상각도, 채도, 명도, 불투명도), rgba(0, 255, 255, 불투명도)
text-align
글자 정렬 설정 start : 현재 텍스트 줄의 시작 위치에 맞추어 문단 정렬
end : 현재 텍스트 줄의 끝 위치에 맞추어 문단 정렬
left : 왼쪽에 맞추어 문단 정렬
right : 오른쪽에 맞추어 문단 정렬
center : 가운데에 맞추어 문단 정렬
justify : 양 쪽에 맞추어 문단 정렬
match-parent : 부모 요소를 따라 문단 정렬
line-height 줄 간격 설정, 배경의 높이와 동일하게 설정하면 텍스트를 세로 중앙으로 정렬 가능
text-decoration
텍스트에 밑줄이나 취소선 표시 none : 줄을 표시하지 않음
underline : 밑줄 표시
overline : 윗줄 표시
line through : 취소선 표시
text-shadow 텍스트에 그림자 효과 추가, text-shadow: none || 가로거리 세로거리 번짐정도 색상;
text-transfrom
텍스트의 대소문자를 변환 capitalize : 첫 글자를 대문자로 변환
uppercase : 모든 글자를 대문자로 변환
lowercase : 모든 글자를 소문자로 변환
full-width : 가능한 한 모든 문자를 전각 문자로 변환
letter-spacing 글자와 글자 사이의 간격을 조절
word-spacing 단어와 단어 사이의 간격을 조절

목록 스타일

종류 설명
list-style-type
목록에서 불릿이나 번호의 종류 설정
disk
  • 채운 원 모양
circlek
  • 빈 원모양
square
  • 채운 사각형
decimal
  • 1부터 시작하는1 0진수
decimal-leading-zero
  • 앞에 0이 붙는 10진수
lower-roman
  • 로마 숫자 소문자
upper-roman
  • 로마 숫자 대문자
lower-alpha || lower-latin
  • 알파벳 소문자
upper-alpha || upper-latin
  • 알파벳 대문자
none
  • 불릿이나 숫자를 없앰
list-style-image
불릿을 원하는 이미지로 바꿀 수 있음 list-style-image : url(이미지파일경로);
list-style-position
들여쓰기 설정 list-style-postion : insinde - 불릿이나 번호를 기본 위치보다 안으로 들여 씀
list-style-postion : none - 기본 값
list-style
목록 속성을 한번에 표시할 수 있음 예시)
list-style : lower-alpha inside;

표 스타일

종류 설명
caption-side
캡션의 위치 설정 caption-side : top - 기본값, 캡션을 표 윗부분에 표시
caption-side : bottom - 캡션을 표 아랫부분에 표시
border
표 바깥 테두리와 셀 테두리를 각각 지정 표 바깥 테두리 : table 태그에 설정
셀 테두리 : td, th 태그에 설정
border-spacing 셀과 셀 사이의 여백 조절, border-spacing : 수평거리 수직거리
border-collapse
셀과 셀 사이의 테두리, 셀과 표의 테두리를 합침 border-collapse : collapse - 테두리 합침
border-collapse : seperate - 기본값, 테두리 분리

박스 모델

종류 설명
width 컨텐츠 영역의 너비 지정, 단위 - px, em, %, 기본값: auto
height 컨텐츠 영역의 높이 지정, 단위 - px, em, %, 기본값: auto
box-sizing
패딩과 테두리 영역까지 사이즈 지정에 포함할 것인지 box-sizing : border-box - 테두리까지 포함해서 크기를 지정 box-sizing : content-box - 기본값, 컨텐츠 영역만 크기 지정
box-shadow
박스 모델에 그림자 효과 부여 box-shadow : 수평거리 수직거리 흐림정도 번짐정도 색상 inset
흐림 정도 : 숫자가 클 수록 부드럽고 작을 수록 진함
번짐 정도 : 양숫값을 사용하면 그림자가 박스보다 크게 퍼짐
색상 : 공백으로 구분해서 여러 개의 색상도 지정 가능
inset : 이 키워드를 적어주면 그림자가 외부가 아닌 내부로 표시됨
border-style
테두리 스타일을 지정 none : 기본값, 테두리 없음
hidden : 테두리 감추기, 표에서 테두리가 collapse인 경우 다른 테두리까지 감춤
solid : 실선 테두리
dotted : 점섬 테두리
dashed : 짧은 직선 테두리
double : 이중 선 테두리
groove : 홈이 파인 듯한 테두리
ridge : 튀어나온 듯한 테두리
inset : 창 내부로 들어간 듯한 테두리
outset : 창에서 튀어나온 듯한 테두리
border-width
테두리 두께 지정 thin || medium || thick 혹은 px 단위로 지정 가능
border-color 테두리 색상 지정
border
테두리 스타일을 묶어서 지정 border : 두께 스타일 색상 (순서 상관 없음)
border-radius 테두리를 둥글게 설정, border-radius: 크기 || 백분율 (타원형일 경우 /입력 후 세로 반지름 길이도 추가)
margin
요소 주변의 여백을 설정 px, em, % 등 수치 또는 부모 요소 기준 백분율로 표시
auto: 자동 설정, 좌우 마진을 auto로 지정할 경우 요소를 가운데로 정렬
세로 마진의 경우 위, 아래 요소의 마진값은 서로 더해지는 것이 아니라 큰 값으로 중첩됨
padding 컨텐츠 영역과 테두리 사이의 여백
diplay
요소의 배치 방법을 결정 block : 블럭 레벨 요소로 만듦
inline : 인라인 레벨 요소로 만듦
inline-block : 인라인, 블럭 레벨 요소의 속성을 모두 가지며 마진과 패딩 지정 가능
none : 해당 요소를 화면에 표시하지 않음
float
요소를 왼쪽이나 오른쪽으로 배치 float : left - 해당 요소를 문서의 왼쪽에 배치
float : right - 해당 요소를 문서의 오른쪽에 배치
float : none - 기본값, 좌우 어느 족에도 배치하지 않음
clear
float 속성을 해제 float 속성을 사용하면 해당 요소의 뒤에 오는 요소에도 float 속성이 적용되는데 이를 해제할 수 있는 기능
clear : left - float:left를 해제
clear : right - float:right를 해제
clear : both - float을 모두 해제
left 기준 위치의 왼쪽에서부터 요소가 얼마나 오른쪽으로 떨어진 거리
right 기준 위치의 오른쪽에서부터 요소가 얼마나 왼쪽으로 떨어진 거리
top 기준 위치의 위쪽에서부터 요소가 얼마나 아래쪽으로 떨어진 거리
bottom 기준 위치의 아래쪽에서부터 요소가 얼마나 위쪽으로 떨어진 거리
position
요소의 배치 방법 지정 static : 기본값, 문서의 흐름에 맞춰 배치
relative : 위칫값을 지정할수 있음, static과 비슷
absolute : relative값을 사용한 상위 요소를 기준으로 위치를 지정해 배치
fixed : 브라우저 창을 기준으로 위치 지정

배경

종류 설명
background-color 배경색 지정
background-clip
배경색의 지정 범위 설정 border-box : 기본값, 테두리까지 배경 지정
padding-box : 테두리를 뺀 패딩 범위까지 배경 지정
content-box : 컨텐츠 부분에만 배경 지정
background-image 배경 이미지 지정, background-image: url('이미지파일경로');
background-repeat
배경 이미지의 반복 방식 설정 repeat : 기본값, 가득 찰 때까지 가로와 세로로 반복
repeat-x : 가로로 가득 찰 때까지 가로로 반복
repeat-y : 세로로 가득 찰 때까지 세로로 반복
no-repeat : 한 번만 표시하고 반복하지 않음
background-position
배경 이미지의 위치 설정 background-position : 수평위치 수직위치;
수평위치: left || center || right || 백분율 || 길이 값
수직위치: top || center || botton || 백분율 || 길이 값
background-origin
배경 이미지의 적용 범위 설정 content-box: 컨텐츠 부분에만 배경이미지 표시
padding-box: 패딩까지 배경이미지 표시
border-box: 테두리까지 배경이미지 표시
background-attachment
스크롤할 때 이미지가 움직일지 고정될지를 설정 scroll: 기본값, 화면을 스크롤하면 배경 이미지도 스크롤됨
fixed: 화면을 스크롤하면 배경 이미지는 고정되고 내용만 스크롤됨
background
background 속성 하나로 배경이미지관련 속성 제어 가능 예시)
background: url('이미지파일경로') no-repeat center bottom fixed;
background-size
배경 이미지 크기 설정 auto: 기본값, 원래 배경이미지 크기만큼 표시
contain: 요소 안에 배경 이미지가 다 들어오도록 이미지를 확대/축소
cover: 배경 이미지로 요소를 모두 덮도록 이미지를 확대/축소
크기: 이미지의 너비와 높이를 px 등의 단위로 지정, 값이 하나만 주어질 경우 너비로 인식하여 높이를 자동 계산
백분율: 요소의 크기를 기준으로 배경 이미지의 크기 조절
linear-gradient
직선적인 그라데이션을 가지는 색상 linear-gradient(to 방향또는각도, 시작색상, 중지점에서색상과위치(%), ,,, , 끝색상);
예시)
linear-gradient(to right bottom, blue, white);
linear-gradient(to 45deg, #06f, white 30%, black);
radial-gradient
타원형 또는 원형의 그라데이션을 가지는 색상 radial-gradient(형태 크기 위치, 시작색상, 중지점에서색상과위치(%), ,,, , 끝색상);
크기: closest-side - 그라데이션 가장자리가 가장 가까운 모서리에 닿는 크기
closest-center - 그라데이션 가장자리가 가장 가까운 코너에 닿는 크기
farthest-side - 그라데이션 가장자리가 가장 먼 모서리에 닿는 크기
farthest-corner - 그라데이션 가장가리가 가장 먼 모서리에 닿는 크기

예시)
radial-gradient(white, yellow, red);
radial-gradient(circle, #06f, white 30%);
radial-gradient(circle closest-side at 20% 20%, #06f, white 30%);
repeating-linear(radial)-gradient
그라데이션의 반복 예시)
repeating-linear-gradient(yellow, red 20px);
repeating-radial-gradient(circle, white, white 10%, #ccc 10%, #ccc 20%);

종류 설명

애니메이션 이름, 바뀌는 지점, 바뀌는 속성을 지정 @keyframes 이름 {

}
지점의 경우 스타일이 바뀌는 지점을 의미, from과 to 혹은 0%, 50%, 100% 등으로 설정 가능

reverse : 애니메이션을 반대로 진행
alternate : 홀수 번째는 normal, 짝수 번째는 reverse 방향으로 진행

animation-duration 애니메이션의 진행 시간

반응형 웹

종류 설명
<meta>
뷰포트를 지정하는 태그, <head>태그 내에 작성
기본형: <meta name="viewport" content="속성1=값1, 속성2=값2, ...">
종류 설명 사용가능한 값 기본값
width 뷰포트 너비 device-width || 크기 브라우저 기본값
height 뷰포트 높이 device-height || 크기 브라우저 기본값
user-scalable 확대/축소가능 여부 yes || no yes
initial-scale 초기 확대/축소값 1 ~ 10 1
@media
특정 미디어에서 어떤 CSS를 적용할 것이지를 지정, <style>태그 안에서 사용
기본형: @media only||not 미디어유형 and (조건) and (조건) ... { }

only : 미디어 쿼리를 지원하지 않는 웹 브라우저에서는 실행하지 않음
not : 다음에 지정하는 미디어 유형을 제외한 미디어에만 적용함

미디어 유형 종류 설명
all 모든 유형
print 인쇄 장치
screen 컴퓨터, 스마트폰
tv TV
aural 음성 합성 장치
braille 점자 표시 장치
handheld 패드와 같은 손에 들고 다니는 장치
projection 프로젝터
tty 디스플레이 기능이 제한된 장치, 픽셀 단위가 사용 불가능함
embossed 점자 프린터
실제 웹 문서 내용이 화면에 나타나는 영역인 뷰표트의 너비와 높이를 조건으로 사용 가능
너비, 높이 속성 종류 설명
width, height 특정 너비, 높이에만 CSS 적용
min-width, min-height 웹 페이지의 최소 너비와 높이 지정, 해당 값 이상의 경우에 CSS 적용
max-width, max-height 웹 페이지의 최대 너비와 높이 지정, 해당 값 이하의 경우에 CSS 적용
단말기의 너비와 높이를 조건으로 사용 가능, 일반적으로 단말기 화면의 해상도와 단말기에서 사용하는 웹 브라우저의 해상도는 서로 다름
단말기의 너비, 높이 속성 종류 설명
device-width, device-height 특정 너비, 높이의 단말기에 CSS 적용
min-device-width, min-device-height 단말기의 최소 너비와 높이 지정, 해당 값 이상의 경우에 CSS 적용
max-device-width, max-device-height 단말기의 최대 너비와 높이 지정, 해당 값 이하의 경우에 CSS 적용
단말기의 화면이 세로방향인지 가로방향인지를 조건으로 사용 가능
단말기의 너비, 높이 속성 종류 설명
orientation: portrait 단말기가 세로모드일 때 CSS 적용
orientation: landscape 단말기가 가로모드일 때 CSS 적용

플렉스

종류 설명
display: flex
<div> 를 플렉스 컨테이너로 설정 flex: 플렉스 컨테이너 내의 플렉스 항목을 블럭 레벨 요소로 배치
inline-flex: 플렉스 항목을 인라인 레벨 요소로 배치
flex-direction
주축의 방향과 플렉스 항목의 배치 순서를 설정 row: 주축을 가로로 지정, 플렉스 항목을 왼쪽에서 오른쪽으로 배치

column: 주축을 세로로 지정, 플렉스 항목을 위에서 아래로 배치

플렉스 항목이 플렉스 컨테이너 범위를 벗어날 때 화면에 어떻게 보이게 할지를 설정 nowrap: 기본값, 플렉스 항목을 한 줄에 표시
wrap: 플렉스 항목을 여러 줄에 표시
wrap-reverse: 플렉스 항목을 여러 줄에 표시, 각 줄은 아래에서부터 위쪽 방향으로 배치됨

flex-end: 주축의 끝점에 맞춰 배치
center: 주축의 중앙에 맞춰 배치
space-between: 첫째항목과 마지막 항목을 주축의 양 끝단에 맞추고 나머지는 일정한 간격을 갖도록 배치
space-around: 모든 항목을 주축에 같은 간격으로 배치, 양 끝부분에 공백이 생김

flex-end: 교차축의 끝점에 맞춰 배치
center: 교차축의 중앙에 맞춰 배치
baseline: 교차축의 문자 기준선에 맞춰 배치
stretch: 플렉스 항목을 늘려 교차축에 가득 차게 배치
align-self
특정 플렉스 항목에 적용되며 교차축(기본적으로 세로)방향의 정렬 방법을 설정 flex-start: 교차축의 시작점에 맞춰 배치
flex-end: 교차축의 끝점에 맞춰 배치
center: 교차축의 중앙에 맞춰 배치
baseline: 교차축의 문자 기준선에 맞춰 배치
stretch: 플렉스 항목을 늘려 교차축에 가득 차게 배치
align-content
플렉스 컨테이너에 적용되며 플렉스 항목이 여러 줄로 표시될 때 각 줄의 교차축(기본적으로 세로)방향으로의 정렬 방법을 설정 flex-start: 교차축의 시작점에 맞춰 배치
flex-end: 교차축의 끝점에 맞춰 배치
center: 교차축의 중앙에 맞춰 배치
space-between: 첫째항목과 마지막 항목을 교차축의 양 끝단에 맞추고 나머지는 일정한 간격을 갖도록 배치
space-around: 모든 항목을 교차축에 같은 간격으로 배치, 양 끝부분에 공백이 생김
stretch: 플렉스 항목을 늘려 교차축에 가득 차게 배치

그리드

종류 설명
display
<div> 를 그리드 컨테이너로 설정 gird: 컨테이너 안의 항목을 블럭 레벨 요소로 배치
inline-grid: 컨테이너 안의 항목을 인라인 레벨 요소로 배치

그리드 컨테이너에 설정, 열의 개수와 너비를 지정 grid-template-columns : 200px 200px 200px; - 너비 200px의 열 3개로 지정
fr : fraction, 그리드에서 사용되는 단위로 Afr Bfr Cfr = A:B:C의 길이비를 가짐, 열의 너비 지정 등에 사용가능

grid-row-gap 행과 행 사이의 간격을 지정
grid-gap 열과 행 사이의 간격을 한번에 지정, grid-gap: 행간격 열간격;
grid-column-start 그리드의 열의 개수가 n개이면 열을 구분하는 세로선은 n+1개가 됨, 그리드 항목이 왼쪽에서부터 몇 번째 세로선의 바로 오른쪽에 위치할 것인지를 설정
grid-column-end 그리드 항목이 몇 번째 세로선까지 자리를 차지할지를 설정, 기본값은 항목의 시작위치 +1
grid-column: n/m 그리드 항목이 몇 번째 세로선부터 몇 번째 세로선까지 자리를 차지할지를 설정, /로 시작선과 끝선 구분
grid-row-start 그리드의 행의 개수가 n개이면 행을 구분하는 가로선은 n+1개가 됨, 그리드 항목이 위쪽에서부터 몇 번째 가로선의 바로 아래쪽에 위치할 것인지를 설정
grid-row-end 그리드 항목이 몇 번째 가로선까지 자리를 차지할지를 설정, 기본값은 항목의 시작위치 +1
grid-row: n/m 그리드 항목이 몇 번째 가로선부터 몇 번째 가로선까지 자리를 차지할지를 설정, /로 시작선과 끝선 구분
grid-area 그리드 항목에 적용되며 템플릿 이름을 지정할 수 있음
grid-template-area
그리드 컨테이너에 적용되며 "" 안에 템플릿 이름과 . 을 사용하여 그리드 항목을 직접 배치할 수 있음 grid-template-area:
  "box1 box1 box1"
  "box2 box3 box3"
  "box2 . box4"
. 은 빈 공간을 의미함
